# MiniDY
基于go-zero框架完成的字节青训营后端项目-“极简抖音”

# 1 项目简介
1. 使用go-zero微服务框架开发，采用rpc+api架构
2. 代码结构使用App{Cmd(api+rpc)层+model(sql,sqlx)层}功能层+Common(公共组件)层，结构清晰
3. 使用内置Jwt进行Token认证
4. 使用ETCD或Endpoints直连进行服务发现
5. 使用Prometheus进行服务监控
6. 使用Jaeger进行链路追踪
7. 使用grafana对服务指标进行可视化读取
8. 使用Nignx进行网关代理
